java8 par
=========

関数型言語による並列処理のサンプル(Java8) - parallelによる並列処理

利用方法
--------

`$ mvn package`

`$ mvn assembly:single`

によって実行可能なJARを生成し、javaコマンドから実行してください。

環境
----

以下の環境でテストしています。

* CPU:AMD Phenom(tm) II X6 1065T Processor （6コア 2.9GHz）
* Memory:16GB
* OS:Ubuntu 12.04.2 LTS (64bit) 3.5.0-36-generic
* Java:JDK(tm) 8 Early Access Releases 1.8.0-ea-b96
* Apache Maven 3.0.5
